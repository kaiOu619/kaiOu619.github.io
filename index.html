哔哩哔哩客户端（相见恨晚）

相见恨晚啊~~

在没发现B站有客户端之前在电脑上刷B站都是通过网站去刷的，在网站刷的时候总是感觉到些许不爽。因为网站把首页推荐的分类分的太细了，没有在手机端上刷B站的快感。偶然在手机端上刷到一个视频，是展示B站客户端的页面效果和功能的，还在下方的简介中放出了下载地址。于是马上跑去下载到电脑上了。

下面分享一下B站客户端的截图：

1、首页

B站客户端还有暗夜模式，再也不怕在晚上刷B站的时候被后面的白色背景亮瞎眼了~

还有，在客户端的推荐内容是所有分类的集合，不像是web端一样细分推荐内容，我觉得这个很完美！

2、侧边栏

​ 这个侧边栏是三段式的设计，页面UI和按钮动画都很完美。

3、播放页

​ 晚上关灯刷视频，舒胡啊~

好了，客户端的页面效果的截图分享。感兴趣的可自行下载（Windows版噢）！

第一次使用Hexo记录

来自CodeSheep 的分享

第一步安装

1、前往nodejs 官网下载nodejs进行安装
2、在命令行查看nodejs版本（node -v）顺便验证nodejs是否安装成功
3、查看npm版本（npm -v）
以上安装完后下面开始安装hexo框架
hexo安装需要借助npm这个包管理器来进行安装，由于国内的镜像源的速度很慢，所以就在使用之前先利用npm来安装cnpm淘宝源（npm install -g cnpm –registry=https://registry.npm.taobao.org）.
安装完后就用cnpm来正式安装hexo框架（cnpm install -g hexo-cli）。（然后可以用hexo -v来验证和查看hexo的信息）

开始创建博客

先新创建一个空文件夹（mkdir 文件名）,如果在创建博客中发生错误，就把创建的文件删除后重新开始创建文件夹的步骤

创建好后 cd进入文件输入（hexo init）初始化（创建）一个博客，完成后可以在文件夹中发现多了好几个文件，此时就是创建成功了，然后可以输入hexo s启动server，下方有个本地的链接端口地址，复制地址在浏览器打开你就会发现你的博客已经有了，它还为你自动生成了一篇博文了。
如果想自己新建一篇博文则命令（hexo n “博文名称”），运行成功后文件就会生成在（博客根目录/source/_posts）文件下，然后就可以写文章了。
写完文章后需要退回博客根目录下，就可以上传你的博客了，首先运行hexo clean清理一下，再来输入hexo g 生成。运行完成就可以hexo s（启动server预览）。到这里hexo博客基本就搭建完成了。

博客部署

这里用到github的仓库来进行部署。
首先登陆到github新建一个仓库（点击右上角加号New repository）。注意：仓库名字（Repository name）命名方式为"昵称.github.io", 仓库名命名方式必须遵循格式。

然后输入Description描述就可以，完成后可以在下方Https|ssh的右侧方框中看到仓库的地址。
建好后我们还需要在博客的根目录下安装一个 git 的插件（cnpm install –save hexo-deployer-git “安装过程中有一个warning的警告可以不用管他”）。
安装完成后需要对博客根目录下的_config.yml文件进行配置，打开_config.yml文件移动到文件的最底部的#Deployment下方输入配置参数，把type改成 type：‘git’，type下方添加repo：仓库地址，branch: master，保存退出。
首先在命令行输入git config –global user.email.”XXX@qq.com“和git config –global user.name.”username”，然后再运行hexo d部署。完成部署后就可以用浏览器输入github的仓库名浏览博客了。

更换主题

更换主题的话就需要自己去找，这里推荐一个（https://github.com/litten/hexo-theme-yilia），用git下载主题 (git clone https://github.com/litten/hexo-theme-yilia themes/文件名），

下载好后配置_config.yml中的theme。修改完就时配置完成了，然后就可以启动server预览一下，用hexo d部署到仓库。完成！！！
